Help

Simple explanations for non-technical users.

What is hosting?

Hosting is where your website “lives” online so people can open it using a link.

What is a domain?

A domain is your website name, like yourbusiness.co.ke. You can buy it later.

How do I go live for free?

Go to Publish → choose Free Hosting → download your HTML file → upload using drag & drop.

Can I update my site later?

Yes. Edit details in the Builder, download again, and upload the new file.
